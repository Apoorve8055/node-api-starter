# Node Api Starter
### **Npm packages dependencies:**
 1. cors
 2. dotenv
 3. express
 4. helmet
 5. jsonwebtoken
 6. morgan
 7. swagger-ui-express
 8. eslint
 9. mocha
 10. nodemon
 11. supertest
 ## Before using

-   Please make sure that you have:
-   node.js installed ([https://nodejs.org/](https://nodejs.org/))
## Usage

To run the project, please use a command line the following:

### Setup

```
npm install
```
### Lint

```
npm run lint
```
### Test

```
npm run test

```
### Development
```
npm run dev
```
